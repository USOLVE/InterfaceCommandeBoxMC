⚠️ Contenu mixte HTTPS / HTTP Cette page est servie en HTTPS mais l'ESP32 répond en HTTP. Le navigateur bloque ces requêtes par défaut.

Sur Chrome mobile :
  1. Tape chrome://flags/#unsafely-treat-insecure-origin-as-secure dans la barre d'adresse.
  2. Ajoute http://<IP_ESP32> dans le champ.
  3. Redémarre Chrome et recharge cette page.

Ou utilise Chrome sur PC → icône cadenas → Paramètres du site → autoriser le contenu non sécurisé.
